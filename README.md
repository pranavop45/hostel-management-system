## 🚀 Overview

PRANAV Hostel Management System is a modern JavaFX-based desktop application designed to manage hostel operations efficiently.

This version is **fully functional without any database**, using local storage to simulate real-time operations like Add, Update, Delete, and Fees management.

---

## ✨ Features

* 👨‍🎓 Student Management (Add / Update / Delete)
* 💰 Fees Management System
* 🏢 Hostel Block Allocation (B1, B2, B3, B4)
* 🔍 Search Functionality
* 📊 Live Table View Updates
* 💾 Local Data Persistence (No MySQL required)
* ⚡ Fast & Responsive UI

---

## 🧠 Tech Stack

* Java 17
* JavaFX 21
* Maven
* FXML (UI Design)
* Local JSON Storage (No Database)

---

## ⚙️ System Requirements

Make sure the following are installed:

### 1. Java (JDK 17)

Check:

```
java -version
```

---

### 2. Maven

Check:

```
mvn -version
```

## ▶️ How to Run (IMPORTANT)

### 🔥 Method 1 (Recommended)

1. Open terminal in project folder
2. Run:

```
mvn clean javafx:run
```

---

### ⚠️ If error occurs:

```
mvn clean install
mvn javafx:run
```
